# javascript-conditionals-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- Give 6 examples of comparison operators.: Greater than, less than, equal to, not equal to, greater than or equal to, less than or equal to.

- What data type do comparison expressions evaluate to?: Boolean

- What is the purpose of an `if` statement?: If statements evaluate a condition and if the condition is met, they will execute a code block

- Is `else` required in order to use an `if` statement?: No

- Describe the syntax (structure) of an `if` statement.: The if keyword, a condition, and a statement.

- What are the three logical operators?: || or, && and, ! not

- How do you compare two different expressions in the same condition?: Using else if

- What is the purpose of a `switch` statement?: Uses the strict equality comparison to evaluate an expression to determine if values match and code should run.

- Is a `default` clause required in order to use a `switch` statement?: No, it is optional
